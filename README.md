Email Bomber using SMTP!!
    SMTP stands for Simple Mail Transfer Protocol and this server is used to send mails to users.  

Commands to Install -

    For Turmux Users - 
        pkg install git
        pkg install python
        git clone https://github.com/HackWithAJ/EmailBomber.git

    For Linux Users - 
        git clone https://github.com/HackWithAJ/EmailBomber.git
        cd EmailBomber
        python3 EmailBomber.py
